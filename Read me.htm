Opening and running the program

1. Extract "Robot Simulator.Zip" file on your Desktop
2. Click on "Robot Simulator" solution file or .csproj file and load the solution in Visual studio
3. Press "Ctrl + F5" and run the program in "Chrome" browser

How to use the Program

1. Enter the Command in the similar way as mentioned in the list (commands are case in-sensitive)
2. To place the Robot on table, Place command is required
3. "Back to Origin" button, will place the Robot at Origin position (0,0)
4. "Reset" button will remove the Robot from Table
